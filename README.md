# 📝 To-Do List Application

A simple and responsive **To-Do List Web Application** built using **HTML**, **CSS**, and **JavaScript**. This project helps users manage their daily tasks with an easy-to-use interface.

## 🚀 Live Demo

🌐 **Website:** https://assignment3-todo.vercel.app/

## ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ⌨️ Add tasks by pressing the **Enter** key
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
